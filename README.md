# PSE ESKAVI - Frontend

This is the frontend vue app for the PSE ESKAVI Web-App

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
